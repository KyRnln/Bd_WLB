# 商务WLB插件（Chrome 扩展）

面向通用输入场景的轻量快捷短语工具：在任意网页输入框/富文本内，输入 `/` 唤出浮动选择框，实时筛选预设短语，方向键/鼠标选择，Enter 插入并自动移除触发符号。

当前版本：**v1.0.6.1**

## 功能概览
- **快捷插入**：输入 `/` 即弹出短语列表，支持上下键/鼠标悬停选中，Enter 插入。
- **实时筛选**：依据光标后方文本即时过滤（标题或内容命中），无匹配时提示“暂无匹配短语”。
- **自动清理触发符**：插入后自动移除 `/`，光标定位在短语末尾。
- **弹窗管理**：在 popup 中新增、编辑、删除短语，列表即时刷新。
- **状态提示**：操作结果在弹窗内提示；浮窗位置自动防溢出（贴边回缩）。
- **数据存储**：使用 `chrome.storage.local`，无需额外后端。
- **订单查询**：通过订单ID查询达人履约情况并导出CSV，支持多订单ID输入，显示导出进度。
- **达人管理**：CSV 导入达人ID，页面自动匹配高亮，支持搜索和编辑。

## 安装与加载
1) 打开 `chrome://extensions/`，开启右上角「开发者模式」。  
2) 点击「加载已解压的扩展程序」，选择本项目根目录（含 `manifest.json`）。  
3) 右上角扩展图标可打开弹窗进行短语管理。

## 使用说明
- 在任意输入框、`textarea` 或可编辑区域输入 `/`，即刻弹出短语列表。
- 输入更多字符（光标后方）可实时缩小匹配范围。
- 上/下键或鼠标悬停高亮，Enter 插入；Esc 关闭。
- 插入后触发的 `/` 会被自动删除。

## 达人导入模板（CSV）
在扩展弹窗的「达人管理」中支持 CSV 导入，**匹配使用 `creator_id` 完全相同**（用于页面高亮）。`creator_cid` 目前仅做数据预留（后续迭代使用）。

模板（带表头，推荐）：

```csv
creator_id,creator_cid
ameera_zainal,1234567890
```

说明：
- 支持列名：`creator_id` / `creator_cid`（或简写 `id` / `cid`）
- 列顺序不限（会按表头识别）

```bash
├── background.js     # 预留的后台脚本

└── README.md         # 说明文档
```

## 调试提示

- 浮窗：页面 DevTools Console 查看 `content.js` 日志，必要时重新加载扩展。
