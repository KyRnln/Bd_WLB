# TikTok视频封面获取功能

## 功能概述

批量获取TikTok视频的封面图片链接，支持导出包含封面图片的Excel文件。

## 核心流程

### 封面获取流程

```
1. 用户输入TikTok视频链接（每行一个）
2. 点击"获取封面"按钮
3. 对每个链接调用 TikTok oEmbed API
4. 解析返回的JSON数据，提取 thumbnail_url
5. 显示封面预览
6. 支持导出包含封面图片的Excel
```

## 组件详解

### 前端 (cover.js)

**主要功能：**

```javascript
// 验证TikTok URL
isValidTikTokUrl(url)

// 获取封面信息
fetchTikTokCover(videoUrl)

// 导出Excel
exportCoverExcel()
```

**TikTok oEmbed API：**

```
API端点: https://www.tiktok.com/oembed?url={videoUrl}

返回格式:
{
  "thumbnail_url": "https://...",
  "title": "视频标题",
  "author_name": "作者名"
}
```

## 数据格式

```javascript
{
  url: 'https://www.tiktok.com/@user/video/123',
  thumbnailUrl: 'https://p16-sign.tiktokcdn.com/...',
  title: '视频标题',
  author: '作者名',
  status: 'success' | 'error',
  error: '错误信息',
  errorDetail: '详细错误信息'
}
```

## 错误处理

| 错误类型 | 说明 | 处理方式 |
|----------|------|----------|
| 无效格式 | URL格式不正确 | 标记失败，显示错误 |
| 请求超时 | API响应超过10秒 | 标记失败，提示检查网络 |
| 地区限制 | TikTok在当前地区不可用 | 标记失败，提示使用VPN |
| CORS错误 | 跨域请求被阻止 | 标记失败，显示详情 |
| JSON解析失败 | 返回非JSON格式 | 标记失败，显示详情 |

## 并发控制

- 同时最多3个并发请求
- 批量处理，避免过多请求

## 使用要求

- 需要能够访问 TikTok 服务
- 部分地区可能需要使用VPN

## 常见问题

### Q: 为什么获取失败？

A: 可能的原因：
1. 当前地区无法访问TikTok（需要VPN）
2. 视频链接已失效
3. 网络连接问题

### Q: 如何解决地区限制？

A: 使用VPN切换到美国、日本、新加坡等地区后重试。
