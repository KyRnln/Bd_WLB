# 通过CID查达人功能

## 功能概述

通过达人CID批量查询对应的达人ID（username），并导出包含头像的Excel文件。

## 核心流程

### 单个CID查询流程

```
1. 用户输入CID
2. 后台打开达人详情页（URL包含CID）
3. 内容脚本从页面提取达人ID和头像URL
4. 返回结果给后台
5. 保存结果并关闭标签页
6. 继续处理下一个CID
```

### 批量处理流程

```
1. 用户输入多个CID（每行一个）
2. 点击"开始查询"按钮
3. 后台创建批量查询任务
4. 依次处理每个CID：
   a. 打开达人详情页
   b. 等待页面加载
   c. 提取达人ID和头像
   d. 关闭标签页
   e. 更新进度显示
5. 全部完成后导出Excel
```

## 组件详解

### 1. 前端 (cid_to_name.js)

**职责：**
- 用户交互界面
- 批量任务状态轮询
- 结果展示与导出

### 2. 后台模块 (cid_to_name_background.js)

**职责：**
- 批量任务调度
- 标签页生命周期管理
- 数据存储与导出

**消息处理：**

| Action | 说明 |
|--------|------|
| `startBatchQuery` | 启动批量查询 |
| `getQueryStatus` | 获取查询状态 |
| `stopBatchQuery` | 停止批量查询 |
| `exportResults` | 导出结果 |

### 3. 内容脚本 (cid_to_name_content.js)

**职责：**
- 从达人详情页提取达人ID
- 从页面提取头像URL

**核心函数：**

```javascript
// 提取达人ID
function extractUsername() {
  // 多种选择器尝试
  const selectors = [
    'span[data-e2e="b7f56c3b-f013-3448"]',
    '[data-e2e*="username"]',
    '.text-head-l.mr-8',
    // ...
  ];
  // 遍历选择器找到有效用户名
}

// 验证用户名格式
function isValidUsername(text) {
  return text.length >= 3 && text.length <= 50 &&
    /^[a-zA-Z0-9_@.-]+$/.test(text) && /[a-zA-Z]/.test(text);
}
```
