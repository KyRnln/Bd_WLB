# 订单查询功能

## 功能概述

通过订单ID批量查询订单详情，自动抓取达人ID、产品ID、订单状态等信息，并导出Excel文件。

## 组件详解

### 1. 前端 (order.js)

**职责：**
- 用户交互界面
- 批量任务状态轮询
- 结果展示与导出

### 2. 后台模块 (order_background.js)

**职责：**
- 批量任务调度
- 标签页生命周期管理
- 数据导出

**消息处理：**

| Action | 说明 |
|--------|------|
| `startBatchOrderQuery` | 启动批量查询 |
| `getBatchOrderStatus` | 获取批量查询状态 |
| `stopBatchOrderQuery` | 停止批量查询 |
| `exportOrderData` | 导出订单数据 |

### 3. 内容脚本 (order_content.js)

**职责：**
- 页面自动化操作
- 表格数据抓取
- IndexedDB存储

## 自动化流程详解

```
1. 导航到样品申请页面
2. 点击"全部"标签
3. 点击搜索字段选择框
4. 选择"订单ID"选项
5. 在搜索框输入订单ID
6. 触发Enter键或点击搜索按钮
7. 等待表格数据加载
8. 解析表格行，提取订单信息
9. 按订单ID分组数据
10. 保存到IndexedDB
```

## 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| 元素未找到 | 尝试备选选择器 |
| 页面加载超时 | 继续执行，可能获取空数据 |
| 数据保存失败 | 抛出错误，停止当前订单处理 |
| 搜索触发失败 | 尝试点击搜索按钮 |

## 性能优化

1. **智能选择器**：多种备选选择器，提高元素查找成功率
2. **分组处理**：按订单ID分组，支持多产品订单
3. **增量存储**：每条数据立即保存，避免数据丢失
4. **进度显示**：实时显示查询进度
